Чтобы просмотреть информацию по угрозе - кликните по соответсвующей записи.
Кнопка "Сохранить" позволяет осуществить сохранение в файл thrlist.xlsx внутри проекта,
кнопка "Сохранить как" - в любом месте.
Навигация по страницам - кнопки "<-" и "->" 
"Обновление" скачивает последнюю версию с сайта ФСТЭК и выводит отчёт.
